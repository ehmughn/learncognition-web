# Web (React) Project Instructions

## Standards & Conventions
- **Framework**: React 19 + Vite.
- **Language**: JavaScript (with some TypeScript types for IDE support).
- **Styling**: Prefer Vanilla CSS (see `App.css`, `index.css`).
- **Icons**: Use `lucide-react`.
- **Routing**: Router-based rendering in `App.jsx` via `RouteRenderer`.

## Teacher Module
- Follow the specifications in `mobile/teacher-module-react-prompt.md` for the teacher dashboard.
- Data model must align with `profiles`, `teachers`, `modules`, `sections`, and `activities` tables.

## Build & Test
- Dev: `npm run dev`
- Lint: `npm run lint`
- Build: `npm run build`
